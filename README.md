# 🧬 Microplásticos en Primates — Dashboard Interactivo

Visualización interactiva de microplásticos en primates, explorando su presencia en diferentes condiciones, especies e individuos.

## 🔍 ¿Qué muestra este proyecto?

- Comparación de microplásticos entre vida silvestre y cuidado profesional  
- Distribución por especie  
- Exploración interactiva por individuo  
- Diferenciación entre tipos de microplástico (fibra y fragmento)  

## 🌐 Ver proyecto en línea

👉 https://zavalaby16.github.io/microplastics-dashboard/

## 🧠 Interpretación

Los microplásticos están presentes tanto en vida silvestre como en individuos bajo cuidado profesional, lo que sugiere una exposición extendida en distintos entornos.  
Estos datos representan cantidades observadas y no implican diferencias estadísticamente significativas.

## 🛠️ Tecnologías utilizadas

- JavaScript (Plotly, D3.js)  
- HTML5  
- CSS3
